# Astro-Olympiad-from-Scratch
Welcome to this repository! This project contains a comprehensive, self-made guide on Celestial Mechanics designed to help students prepare for Astronomy and Astrophysics Olympiads.I created this resource during my own preparation for the Bangladesh Olympiad on Astronomy and Astrophysics (BDoAA) Team Selection Test (TST). It is completely public so that future contestants can benefit from it.📚 

About the Resource Topic: Celestial Mechanics (Core concepts, derivations, and problem-solving techniques).

Format: Comprehensive PDF guide.

Target Audience: High school students preparing for national Olympiads (BDoAA) and international selections (IOAA).
